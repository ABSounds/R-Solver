Wave Digital Filter R-Type Solver

This is a simple web app to derive an R-Type scattering matrix to be used in a Wave Digital Filter circuit model.

To use it, upload a netlist for the R-Type adapter that you would like to solve.

Then choose a datum node (default to 0), and a node to adapt (default to -1/unadapted), and click solve to derive the scattering matrix.
